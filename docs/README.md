# AgentManager 文档中心

## 目录

| 目录 | 内容 |
|------|------|
| `architecture/` | 系统架构、数据流、设计决策 |
| `guides/` | 安装、开发、调试操作手册 |
| `plans/` | 实施计划（按日期组织，先写 plan 再动手） |

## 快速导航

- **第一次上手？** → `guides/setup.md`
- **想了解架构？** → `architecture/overview.md`
- **要做新功能？** → 先在 `plans/YYYY-MM-DD/<topic>.md` 写 plan
- **AI 工作协议** → 根目录 `AGENTS.md`
- **模块边界** → `src-tauri/SPEC.md` / `src/SPEC.md`
